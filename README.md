# shadhq - Personal Blog Website

## Tech stack
**Astro 5**, **Tailwind CSS v4**, and **Markdown**.

## Development

```bash
npm install
npm run dev
```
## Adding Blog Posts
Create a new Markdown file (`.md`) inside the `src/content/blog/` directory.

Use this Frontmatter template:
```markdown
---
title: "Your Post Title"
description: "Brief summary."
pubDate: 2026-03-24
tags: ["tag1", "tag2"]
---
```
